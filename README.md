# belt-clip
# Using package
##Installing
```
    npm install --save belt-vue-components
```
##Using in project
```
ESM
import 'belt-vue-components'

Common/UMD
require ('belt-vue-components')
```
###Syntax
```
    <belt-clip :attachment="object" :recipe="[{screens: 'default', height: 960, width: 1440}]">
```
